# Snake Smart 

Deep learning-based Indian snake classification system using TensorFlow, Keras, and OpenCV.

This project was developed to identify different species of Indian snakes from images using a Convolutional Neural Network (CNN) with VGG16 transfer learning.

---

# Features

- Snake image classification
- Deep learning using VGG16
- Image preprocessing with OpenCV
- Data augmentation using ImageDataGenerator
- Transfer learning approach
- Google Drive + Local PC dataset support
- TensorFlow/Keras implementation
